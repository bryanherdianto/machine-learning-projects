# Machine Learning Projects

A collection of machine learning and deep learning projects — from building a neural network from scratch with NumPy to training CNNs, LSTMs, and KNN models with TensorFlow and scikit-learn.

## Projects

| Project                                                             | Topic                                                       | Library       |
| ------------------------------------------------------------------- | ----------------------------------------------------------- | ------------- |
| [Artificial Neural Network](./artificial-neural-network/)           | ANN from scratch (NumPy only) — classifies a spiral dataset | NumPy         |
| [Rock Paper Scissors](./boilerplate-rock-paper-scissors/)           | Markov chain player that learns opponent patterns           | Python stdlib |
| [Book Recommendation](./book-recommendation/)                       | KNN with cosine distance for book recommendations           | scikit-learn  |
| [Cat and Dog Classifier](./cat-dog-classifier/)                     | CNN that classifies images of cats and dogs                 | TensorFlow    |
| [Health Cost Regression](./predict-health-costs-with-regression/)   | Neural network that predicts annual healthcare costs        | TensorFlow    |
| [SMS Text Classification](./sms-text-classification/)               | Bidirectional LSTM that classifies SMS as ham or spam       | TensorFlow    |
| [Analisis Sentimen Pilpres 2024](./analisis-sentimen-pilpres-2024/) | RNN vs LSTM vs GRU for Indonesian election sentiment        | TensorFlow    |

Each folder has its own `README.md` with detailed, beginner-friendly documentation and a `requirements.txt` for dependencies.
